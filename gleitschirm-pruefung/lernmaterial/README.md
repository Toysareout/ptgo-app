# Lernmaterial

Hier liegt dein hochgeladenes Lernmaterial, nach Themen sortiert. Schick mir
Texte, Fotos oder Skripte – ich lege sie hier strukturiert ab und nutze sie
als **einzige Quelle** für meine Antworten.

## Ablage-Konvention

- Pro Themengebiet eine eigene Datei, z. B.:
  - `aerodynamik.md`
  - `meteorologie.md`
  - `geraetekunde.md`
  - `luftrecht.md`
  - `flugpraxis.md`
  - `navigation.md`
  - `erste-hilfe.md`
- Originalbilder/-scans im Unterordner `bilder/` (Dateiname = Thema + Nummer).

> Noch kein Material hinterlegt.
