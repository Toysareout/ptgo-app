Das Lebenswerk. Aus dem Manifest wird ein zusammenhängendes Buch mit einer Stimme —
ein Werk, das Alex selbst liest und das Rosalie eines Tages lesen kann.

## Aufbau in einem Bild

```
OUVERTÜRE  →  Ⅰ Wer bin ich?  →  Ⅱ Wer will ich werden?
           →  Ⅲ Was muss ich tun?  →  Ⅳ Wie lebe ich jeden Tag?
           →  Ⅴ Was bleibt von mir?  →  VERMÄCHTNIS
```

Die zehn Kodizes (PTGO, Gesundheit, Unternehmer, Finanz, Musik, Paragliding,
Freiheit, Beziehung, Krise, Vater) sind in diese Reise eingewoben — jeder dort, wo
er in die innere Entwicklung gehört.

## Die eine Frage

> Wie wird ein Mann zu sich selbst?
> Nicht Motivation. Nicht Mindset. Identität.
> Das Ziel ist nicht, wie jemand anderes zu werden — sondern der beste Alex.
