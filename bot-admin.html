Twilio Webhook URL

    

      https://thetoysareout.com/.netlify/functions/whatsapp-bot
    

    
Diese URL in Twilio → WhatsApp Sandbox → "When a message comes in" eintragen.

  


  

    
Stripe Webhook URL

    

      https://thetoysareout.com/.netlify/functions/bot-stripe-webhook
    

    
Diese URL in Stripe → Developers → Webhooks eintragen. Event: checkout.session.completed

VariableBeschreibung

      
        
SUPABASE_URLSupabase Project URL

        
SUPABASE_SERVICE_ROLE_KEYSupabase Service Role Key (Backend)

        
TWILIO_ACCOUNT_SIDTwilio Account SID

        
TWILIO_AUTH_TOKENTwilio Auth Token

        
TWILIO_WHATSAPP_FROMWhatsApp Nummer (whatsapp:+49...)

        
OWNER_WHATSAPPDeine WhatsApp Nummer für Alerts

        
ANTHROPIC_API_KEYClaude API Key für Bot-Brain

        
STRIPE_SECRET_KEYStripe Secret Key

        
STRIPE_WEBHOOK_SECRETStripe Webhook Signing Secret
